# 我的餐廳清單
## 介紹
紀錄自己的餐廳清單
### 功能
+ 可查看所有餐廳
+ 可瀏覽餐廳詳細資訊
+ 可依關鍵字搜尋特定餐廳
## 開始使用
1.確認有安裝 node.js 與 npm
2.將專案 clone 到本機電腦
3.安裝npm套件，於終端機輸入：npm install
4.安裝完畢後，繼續輸入：npm run start
5.如運行順利，終端機會顯示：Express is listening on localhost:3000
6.如要停止使用，請於終端機輸入：ctrl + c
## 開發工具
+ Font-awesome 5.8.1
+ Express 4.16.4
+ Express-handlebars 3.0.0
+ Bootstrap 5.1
+ Node.js v14.16.0
